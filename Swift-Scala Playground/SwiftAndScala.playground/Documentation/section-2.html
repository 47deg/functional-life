Tuples

Do you remember the old and dark ages, when you wanted to perform a calculation on several values in a function and needed to send its pointer references to do so? Do you feel the same shivering feeling thinking about that as I do? Those times are at last (mostly) over, as Swift introduces tuples (really common in other Scala, Haskell, Python...) to Apple's platforms. A tuple is a set of values packed together, even if they're from different types. Tuples are first-class types, so you can use them as parameters or return them in a function.
